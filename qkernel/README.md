# qkernel

The contextuality-kernel extraction package is maintained as its own public repository:

**https://github.com/manuflog/qkernel**  (Apache-2.0)

It implements the odd-$Q$ criterion, commutator-carry bits, minimal-kernel compression, SAT/MaxSAT
export, obstruction-spectrum lookup, and a runnable sequential-measurement hardware-export protocol.
Install: `pip install -e ".[dev]"`.
